Capítulo 8. Seleção de faixa de cores

O objetivo dessa lição é apresentar como selecionar pixels em uma imagem colorida que se situam em uma faixa de cores fornecida. Processos de seleção de cores são bastante comuns em aplicações de visão computacional, tal como segmentação e rastreamento de objetos. Por exemplo, o rastreamento do PlayStation Move é feito através da seleção de uma faixa de cores previamente designada para a esfera luminosa do controle.

O exemplo da Listagem 25, “colorrange.cpp” abre uma imagem fornecida e permite ao usuário selecionar interativamente um retângulo onde haverá a seleção da faixa de cores.

Listagem 25. colorrange.cpp
#include <iostream>
#include <opencv2/opencv.hpp>

cv::Mat image;
bool selectObject = false;
int trackObject = 0;
cv::Point origin;
cv::Rect selection;

cv::Scalar average;
cv::Scalar stdev;

int rangeR = 0;
int rangeG = 0;
int rangeB = 0;

void on_trackbar_R(int value, void* userdata) { rangeR = value; }
void on_trackbar_G(int value, void* userdata) { rangeG = value; }
void on_trackbar_B(int value, void* userdata) { rangeB = value; }

static void onMouse(int event, int x, int y, int, void*) {
  if (selectObject) {
    selection.x = MIN(x, origin.x);
    selection.y = MIN(y, origin.y);
    selection.width = std::abs(x - origin.x);
    selection.height = std::abs(y - origin.y);
    selection &= cv::Rect(0, 0, image.cols, image.rows);
  }
  switch (event) {
    case cv::EVENT_LBUTTONDOWN: {
      origin = cv::Point(x, y);
      selection = cv::Rect(x, y, 0, 0);
      selectObject = true;
      break;
    }
    case cv::EVENT_LBUTTONUP: {
      selectObject = false;
      if (selection.width > 0 && selection.height > 0) {
        trackObject = 1;
      }
      break;
    }
  }
}

int main(int argc, const char** argv) {
  cv::Rect trackWindow;
  cv::Mat mask;
  cv::Mat frame;
  frame = cv::imread(argv[1]);

  average = cv::Scalar(128, 128, 128);
  stdev = cv::Scalar(0, 0, 0);

  cv::namedWindow("Image", cv::WINDOW_NORMAL);
  cv::namedWindow("Selection", cv::WINDOW_NORMAL);
  cv::createTrackbar("R-range", "Image", &rangeR, 255, on_trackbar_R, &rangeR);
  cv::createTrackbar("G-range", "Image", &rangeG, 255, on_trackbar_G, &rangeG);
  cv::createTrackbar("B-range", "Image", &rangeB, 255, on_trackbar_B, &rangeB);
  cv::setMouseCallback("Image", onMouse, 0);
  for (;;) {
    frame.copyTo(image);
    char c = cv::waitKey(10);
    if (c == 27) {
      break;
    }
    if (selectObject && selection.width > 0 && selection.height > 0) {
      cv::Mat roi(image, selection);
      cv::bitwise_not(roi, roi);
    }
    cv::imshow("Image", image);
    if (trackObject) {
      if (!selectObject && selection.width > 0 && selection.height > 0) {
        cv::Mat roi(image, selection);
        cv::meanStdDev(roi, average, stdev);
        rangeR = 3 * stdev[2];
        cv::setTrackbarPos("R-range", "Image", rangeR);
        rangeG = 3 * stdev[1];
        cv::setTrackbarPos("G-range", "Image", rangeG);
        rangeB = 3 * stdev[0];
        cv::setTrackbarPos("B-range", "Image", rangeB);
        std::cout << "Average: " << average << std::endl;
        std::cout << "Range min: "
                  << average - cv::Scalar(rangeB, rangeG, rangeR) << std::endl;
        std::cout << "Range min: "
                  << average + cv::Scalar(rangeB, rangeG, rangeR) << std::endl;
      }
      trackObject = 0;
    }
    cv::inRange(image, average - cv::Scalar(rangeB, rangeG, rangeR),
                average + cv::Scalar(rangeB, rangeG, rangeR), mask);
    cv::imshow("Selection", mask);
  }
  cv::imwrite("selection.png", mask);
  return 0;
}

Para compilar e executar o programa-exemplo colorrange.cpp, salve-o em um diretório juntamente com o arquivo CMakeLists.txt adaptado para o exemplo (Listagem 26, “CMakeLists.txt”), a imagem psmove.png e execute a seqüência de comandos mostrada na Listagem 27, “Compilação e execução do programa colorrange”.

Listagem 26. CMakeLists.txt

project(colorrange VERSION 0.1.0 LANGUAGES C CXX)

find_package(OpenCV REQUIRED)
include_directories(${OpenCV_INCLUDE_DIRS})

add_executable(colorrange colorrange.cpp)
target_link_libraries(colorrange ${OpenCV_LIBS})

$ cd build
$ cmake ..
$ make
$ ./colorrange ../psmove.png

Figura 17. Saída do programa colorrange

A seleção de faixa de cores é feita através da definição de um intervalo de valores para cada canal de cor (vermelho, verde e azul), intervalo este tomado de um retângulo que é desenhado na imagem. A cor média de cada canal para os pixels dentro do retângulo é calculada e a faixa de cores é definida a partir dessa cor, para um intervalo de \(media - 3 \times variancia, media + 3 \times variancia\), onde a variância é calculada para a amostra selecionada no retângulo.

A Figura 18, “Imagem de um controle PlayStation Move com a respectiva região com faixa de cores selecionada.” ilustra um controle PlayStation Move com a esfera luminosa em cor vermelha. Selecionando uma faixa de cores adequada, é possível separar a esfera luminosa do controle do restante da imagem.

Figura 18. Imagem de um controle PlayStation Move com a respectiva região com faixa de cores selecionada.

8.1. Descrição do programa colorrange.cpp

if (selectObject) {
    selection.x = MIN(x, origin.x);
    selection.y = MIN(y, origin.y);
    selection.width = std::abs(x - origin.x);
    selection.height = std::abs(y - origin.y);
    selection &= cv::Rect(0, 0, image.cols, image.rows);
}

Caso a ferramenta esteja operando no modo de seleção de retângulo, o que ocorre quando o botão esquerdo do mouse é pressionado, a variável selection é atualizada com as coordenadas do retângulo selecionado. A operação &= é um operador de interseção de retângulos, que garante que a seleção não ultrapasse os limites da imagem.

switch (event) {
    case cv::EVENT_LBUTTONDOWN: {
        origin = cv::Point(x, y);
        selection = cv::Rect(x, y, 0, 0);
        selectObject = true;
        break;
    }
    case cv::EVENT_LBUTTONUP: {
        selectObject = false;
        if (selection.width > 0 && selection.height > 0) {
            trackObject = 1;
        }
        break;
    }
}

O evento cv::EVENT_LBUTTONDOWN é acionado quando o botão esquerdo do mouse é pressionado pelo usuário. Neste caso, a variável origin é atualizada com as coordenadas do local onde o botão foi pressionado e a variável selection é inicializada com um retângulo de dimensões nulas e será atualizado quando a movimentação do mouse ocorrer. O evento cv::EVENT_LBUTTONUP é acionado quando o botão esquerdo do mouse é liberado. Neste caso, a variável selectObject é desativada e a variável trackObject é ativada caso o retângulo selecionado tenha dimensões positivas. A variável trackObject é utilizada para indicar que a seleção de faixa de cores foi realizada pelo utilizador e que a aplicação deve processar a imagem para ser exibida na janela de seleção.

if (selectObject && selection.width > 0 && selection.height > 0) {
    cv::Mat roi(image, selection);
    cv::bitwise_not(roi, roi);
}

Caso a ferramenta esteja operando no modo de seleção de retângulo, a região selecionada é invertida (negativo) para facilitar a seleção interativa da região pelo usuário.

if (trackObject) {
    if (!selectObject && selection.width > 0 && selection.height > 0) {
        cv::Mat roi(image, selection);
        cv::meanStdDev(roi, average, stdev);
        rangeR = 3 * stdev[2];
        cv::setTrackbarPos("R-range", "Image", rangeR);
        rangeG = 3 * stdev[1];
        cv::setTrackbarPos("G-range", "Image", rangeG);
        rangeB = 3 * stdev[0];
        cv::setTrackbarPos("B-range", "Image", rangeB);
        //...
    }
}

Quando o usuário termina o processo de seleção liberando o botão esquerdo do mouse, é criada uma região de interesse (roi - region of interest) na imagem original, para a qual é calculada a média e a variância dos valores dos canais de cor. Os valores das faixas de cores são calculados como sendo três vezes a variância para cada canal de cor. O fator multiplicador 3 é arbitrário e pode ser ajustado conforme a aplicação.

Os valores das faixas de cores são utilizados para a seleção dos pixels da imagem que se encontram dentro da faixa de cores definida. Perceba que as faixas de R, G e B são atualizadas nos seus respectivos trackbars, de sorte que o usuário pode fazer ajustes finos nas faixas de cores selecionadas.

cv::inRange(image, average - cv::Scalar(rangeB, rangeG, rangeR),
            average + cv::Scalar(rangeB, rangeG, rangeR), mask);

A função cv::inRange é utilizada para selecionar os pixels da imagem que se encontram dentro da faixa de cores definida. A função recebe como parâmetros a imagem original, a cor mínima e a cor máxima da faixa de cores, produzindo na saída a imagem mask, uma imagem cinzenta, onde os pixels selecionados serão armazenados com valores iguais a 255 e os demais com valores iguais a 0.
