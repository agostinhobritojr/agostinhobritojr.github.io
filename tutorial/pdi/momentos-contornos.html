Capítulo 21. Extração de características: Momentos de Hu para contornos

Os momentos de Hu são uma ferramenta bastante útil para descrição de características em processamento de imagens. Detalhe acerca do seu funcionamento são discutidos no Capítulo 20, Extração de características: Momentos de Hu para regiões. Nesta lição, será mostrado como usar os momentos de Hu para extrair características de contornos de componentes conectadas extraídas de uma imagem previamente segmentada por threshold automático.

A imagem de referência para extração de momentos de Hu é mostrada na Figura 52, “Exemplo de imagem para calulo de momentos de Hu.”, e contém três formas geométricas diferentes: buchas, arruelas e parafusos.

Figura 52. Exemplo de imagem para calulo de momentos de Hu.

O exemplo da Listagem 66, “momentos-contornos.cpp” mostra o processo abrir uma imagem fornecida, segmentar por threshold automático, obter os contornos das componentes conectadas e extrair os momentos de Hu de cada uma delas.

Listagem 66. momentos-contornos.cpp
#include <iostream>
#include <fstream>
#include <opencv2/opencv.hpp>

int main(int argc, char** argv){
  cv::Mat image, gray;
  std::ofstream file;
  cv::Moments momentos;

  image= cv::imread(argv[1], cv::IMREAD_GRAYSCALE);

  if(!image.data){
    std::cout << "nao abriu " << argv[1] << std::endl;
    return 0;
  }
  
  file.open("momentos.txt");

  cv::threshold(image, image, 1, 255, cv::THRESH_BINARY + cv::THRESH_OTSU + cv::THRESH_BINARY_INV);

  std::vector< std::vector<cv::Point> > contours;
  std::vector< cv:: Vec4i> hierarchy;

  cv::findContours(image, contours, hierarchy, cv::RETR_EXTERNAL, cv::CHAIN_APPROX_NONE);

  cv::cvtColor(image, image, cv::COLOR_GRAY2BGR);

  int nformas = 0;

  for (size_t i = 0; i < contours.size(); i++){
    if(contours[i].size() < 10){
      continue;
    }
    nformas++;
    momentos = cv::moments(contours[i]);

    cv::Point2f center(momentos.m10/momentos.m00, momentos.m01/momentos.m00);

    double hu[7];
    cv::HuMoments(momentos, hu);
    if(hu[0] > 0)
    {
      cv::drawContours(image, contours, i, cv::Scalar(0, 0, 255), 2);
    }
    else{
      cv::drawContours(image, contours, i, cv::Scalar(0, 255, 0), 2);
    }
    cv::putText(image, std::to_string(i), center, cv::FONT_HERSHEY_SIMPLEX, 0.5, cv::Scalar(0, 0, 0), 8);
    cv::putText(image, std::to_string(i), center, cv::FONT_HERSHEY_SIMPLEX, 0.5, cv::Scalar(255, 255, 255), 1);

    file << i << ", ";
    for (int j = 0; j < 7; j++){
      hu[j] = -1 * copysign(1.0, hu[j]) * log10(abs(hu[j]));
      file << hu[j] << ", ";
    }
    file << std::endl;
	}
  std::cout << "Numero de objetos: " << nformas << std::endl;
  file.close();

  cv::imshow("janela", image);  
  cv::imwrite("contornos-rotulados.png", image);
  cv::waitKey();
  return 0;
}

Para compilar e executar o programa-exemplo momentos-contornos.cpp, salve-o em um diretório juntamente com o arquivo CMakeLists.txt adaptado para o exemplo (Listagem 67, “CMakeLists.txt”), a imagem formas.png e execute a seqüência de comandos mostrada na Listagem 68, “Compilação e execução do programa momentos”.

Listagem 67. CMakeLists.txt
cmake_minimum_required(VERSION 3.0.0)
project(momentos-contornos VERSION 0.1.0 LANGUAGES C CXX)

find_package(OpenCV REQUIRED)

target_link_libraries(momentos-contornos ${OpenCV_LIBS})

$ cd build
$ cmake ..
$ make
$ ./momentos-contornos ../formas.png

21.1. Descrição do programa momentos.cpp

cv::threshold(image, image, 1, 255, cv::THRESH_BINARY + cv::THRESH_OTSU + cv::THRESH_BINARY_INV);

Calcula o threshold automático da imagem fornecida e inverte os valores dos pixels. O parâmetro cv::THRESH_BINARY_INV é utilizado para inverter os valores dos pixels da imagem, pois o procedimento de extração de contornos do OpenCV considera que os pixels de interesse são brancos.

cv::findContours(image, contours, cv::RETR_EXTERNAL, cv::CHAIN_APPROX_NONE);

Extrai os contornos das componentes conectadas presentes na imagem. Os parâmetros cv::RETR_EXTERNAL e cv::CHAIN_APPROX_NONE indicam que apenas os contornos externos das componentes conectadas serão extraídos e que nenhum tipo de aproximação será feita nos contornos, ou seja, todos os pixels que compõem o contorno serão extraídos.

if(contours[i].size() < 100){
      continue;
}

// ...

momentos = cv::moments(contours[i]);

cv::Point2f center(momentos.m10/momentos.m00, momentos.m01/momentos.m00);

Calcula os momentos do i-ésimo contorno extraído e calcula o centro de massa do contorno, para posterior marcação na imagem de saída. Perceba que os momentos somente são calculados para contornos com mais de 100 pixels, evitando contornos muito pequenos oriundos de artefatos gerados na segmentação.

double hu[7];
cv::HuMoments(momentos, hu);

Calcula os momentos de Hu do i-ésimo contorno extraído.

for (int j = 0; j < 7; j++){
  hu[j] = -1 * copysign(1.0, hu[j]) * log10(abs(hu[j]));
// ...
}

Normaliza os momentos de Hu para fins de visualização e comparação. Mais detalhes sobre a normalização dos momentos de Hu podem ser encontrados no Capítulo 20, Extração de características: Momentos de Hu para regiões.
