Capítulo 21. Extração de contornos

A extração de contornos é uma técnica utilizada em processamento de imagens e classificação de padrões que permite analisar a forma de um objeto em uma imagem. Algoritmos de extração de contorno opera sobre imagens binarizadas, e visam estrair os pixels que compõem a fronteira de uma componente conectada. A fronteira de um objeto é formada pelos pixels que o separam do restante da imagem.

Nas Figura 52, “Exemplo de imagem binarizada com contornos destacados em vermelho.” e Figura 53, “Detalhe da imagem binarizada.” é possível visualizar exemplos de contornos. A imagem binária é composta de várias regiões separadas em cor branca, com o fundo em preto. Os pixels pintados em vermelho representam o contorno e marcam a fronteira de cada região.

Figura 52. Exemplo de imagem binarizada com contornos destacados em vermelho.
Figura 53. Detalhe da imagem binarizada.

Nesta lição será mostrado como extrair contornos de figuras binarizadas utilizando OpenCV. Os contornos extraídos podem ser utilizados para compor descritores de forma que podem ser utilizados em algoritmos de classificação de padrões.

O programa utilizado como referência nesta lição será

Listagem 66. contornos.cpp
#include <fstream>
#include <iostream>
#include <opencv2/opencv.hpp>

int main(int argc, char** argv) {
  cv::Mat image, gray;
  std::ofstream file;

  image = cv::imread(argv[1], cv::IMREAD_GRAYSCALE);

  if (!image.data) {
    std::cout << "nao abriu " << argv[1] << std::endl;
    return 0;
  }

  cv::threshold(image, image, 1, 255, cv::THRESH_BINARY + cv::THRESH_OTSU);

  std::vector<std::vector<cv::Point> > contours;
  std::vector<cv::Vec4i> hierarchy;

  cv::findContours(image, contours, hierarchy, cv::RETR_EXTERNAL,
                   cv::CHAIN_APPROX_NONE);

  cv::cvtColor(image, image, cv::COLOR_GRAY2BGR);

  file.open("contornos.svg");
  if (!file.is_open()) {
    std::cout << "nao abriu contornos.svg" << std::endl;
    return 0;
  }

  file << "<svg height=\"" << image.rows << "\" width=\"" << image.cols
       << "\" xmlns=\"http://www.w3.org/2000/svg\"> " << std::endl;

  for (size_t i = 0; i < contours.size(); i++) {
    file << "<path d=\"M " << contours[i][0].x << " " << contours[i][0].y
         << " ";
    for (size_t j = 1; j < contours[i].size(); j++) {
      file << "L" << contours[i][j].x << " " << contours[i][j].y << " ";
    }
    file << "Z\" fill=\"#cccccc\" stroke=\"black\" stroke-width=\"1\" />"
         << std::endl;
    cv::drawContours(image, contours, i, CV_RGB(255, 0, 0));
  }
  file << "</svg>" << std::endl;

  cv::imshow("imagem", image);
  cv::waitKey();
  return 0;
}

Para compilar e executar o programa-exemplo contornos.cpp, salve-o em um diretório juntamente com o arquivo CMakeLists.txt adaptado para o exemplo (Listagem 67, “CMakeLists.txt”), a imagem formas.png e execute a seqüência de comandos mostrada na Listagem 68, “Compilação e execução do programa contornos”.

Listagem 67. CMakeLists.txt
cmake_minimum_required(VERSION 3.0.0)
project(contornos VERSION 0.1.0 LANGUAGES C CXX)

find_package(OpenCV REQUIRED)
include_directories(${OpenCV_INCLUDE_DIRS})

add_executable(contornos contornos.cpp)
target_link_libraries(contornos ${OpenCV_LIBS})
Listagem 68. Compilação e execução do programa contornos
$ mkdir build
$ cd build
$ cmake ..
$ make
$ ./contornos ../formas.png

O programa irá gerar um arquivo chamado contornos.svg contendo os contornos externos das componentes conectadas presentes na figura formas.png no formato Scalable Vector Graphics. O arquivo contornos.svg é mostrada na Figura 54, “Exemplo de contornos extraídos da imagem formas.png.”

Figura 54. Exemplo de contornos extraídos da imagem formas.png.

21.1. Descrição do programa contornos.cpp

cv::threshold(image, image, 1, 255, cv::THRESH_BINARY + cv::THRESH_OTSU);

O programa contornos.cpp abre a imagem em escala de tons de cinza e realiza sua binarização, posto que o algoritmo de extração de contornos necessita de uma imagem binária para operar. A binarização é realizada com o método de Otsu, conforme abordado no Capítulo 18, Limiarização.

cv::findContours(image, contours, cv::RETR_EXTERNAL, cv::CHAIN_APPROX_NONE);

Uma vez binarizada, a imagem é encaminhada para a função findcontours(), que extrai os contornos das componentes conectadas usando seguidores de borda. Embora ela consiga montar toda uma hierarquia de contornos internos e externos para componentes conectadas com buracos no seu interior, o programa contornos.cpp está configurado para extrair apenas os contornos externos (cv::RETR_EXTERNAL). Ainda, a findcontours() é configurada para não aproximar os contornos, extraindo portanto todos os pontos que compõem o contorno da componente conectada (cv::CHAIN_APPROX_NONE).

cv::cvtColor(image, image, cv::COLOR_GRAY2BGR);

A imagem é convertida para o espaço de cores BGR de modo que os contornos possam ser desenhados em vermelho no final.

file.open("contornos.svg");
//...
file << "<svg height=\"" << image.rows << "\" width=\"" << image.cols <<
  "\" xmlns=\"http://www.w3.org/2000/svg\"> " << std::endl;
cv::drawContours(image, contours, i, CV_RGB(255, 0, 0));

O programa cria um arquivo contornos.svg e escreve o cabeçalho do arquivo SVG, informando a altura e largura da imagem. Em seguida, no laço que percorre os contornos extraídos, é realizado o desenho dos contornos no arquivo, preenchendo-os com cinza e com traço de borda preto. Os mesmos contornos são também desenhados em vermelho na imagem original.

Perceba que o arquivo no formato svg é criado apenas para fins de visualização após o término do programa. O arquivo svg pode ser aberto em qualquer editor de texto e visualizado em navegadores web, ou utilizando uma ferramenta livre como o Inkscape para editá-lo.
