Capítulo 8. Combinação de Imagens

A combinação de imagens é uma técnica que permite a junção de duas ou mais imagens em uma única imagem usando operações algébricas ou lógicas. A combinação de imagens pode ser feita de várias maneiras, como utilizando adição, subtração, multiplicação ou divisão das imagens e pode ser utilizada para criar efeitos visuais, melhorar a qualidade das imagens ou extrair informações relevantes.

Nesta lição, será mostrado como realizar combinação de imagens usando algumas das principais operações aritméticas e lógicas disponíveis no OpenCV.

O exemplo da Listagem 25, “combinacao.cpp” realiza o processo de abrir duas imagens de mesmo tamanho e combiná-las de diferentes formas.

Listagem 25. combinacao.cpp
#include <iostream>
#include <opencv2/opencv.hpp>

int main(int argc, char** argv) {
  cv::Mat image1, image2, comb;
  cv::Mat img_add, img_sub, img_max, img_min;
  cv::Mat img_addweighted, img_and, img_or, img_not, img_xor;
  cv::Vec3b val;

  if (argc != 3){
    std::cout << "Usage: " << argv[0] << " <imagem1> <imagem2>" << std::endl;
    return -1;
  }

  image1 = cv::imread(argv[1], cv::IMREAD_GRAYSCALE);
  image2 = cv::imread(argv[2], cv::IMREAD_GRAYSCALE);

  if (!image1.data){
    std::cout << "nao abriu " << argv[1] << std::endl;
    return 0;
  } 

  if (!image2.data){
    std::cout << "nao abriu " << argv[2] << std::endl;
    return 0;
  }

  // compare se as imagens tem o mesmo tamanho
  if (image1.size() != image2.size()) {
    std::cout << "As imagens devem ter o mesmo tamanho" << std::endl;
    return 0;
  }

  cv::hconcat(image1, image2, comb);
  cv::imshow("imagens", comb);

  // operacoes aritmeticas
  img_add = image1 + image2;
  img_sub = cv::abs(image1 - image2);
  img_max = cv::max(image1, image2);
  img_min = cv::min(image1, image2);
  cv::addWeighted(image1, 1, image2, 0.3, 0, img_addweighted);

  // concatenando as imagens
  cv::hconcat(img_add, img_sub, comb);
  cv::hconcat(comb, img_max, comb);
  cv::hconcat(comb, img_min, comb);
  cv::hconcat(comb, img_addweighted, comb);
  cv::imshow("aritmeticas", comb);

  // operacoes logicas
  cv::bitwise_and(image1, image2, img_and);
  cv::bitwise_or(image1, image2, img_or);
  cv::bitwise_not(image1, img_not);
  cv::bitwise_xor(image1, image2, img_xor);

  // concatenando as imagens
  cv::hconcat(img_and, img_or, comb);
  cv::hconcat(comb, img_not, comb);
  cv::hconcat(comb, img_xor, comb);
  cv::imshow("logicas", comb);

  cv::waitKey();
  return 0;
}

Para compilar e executar o programa-exemplo combinacao.cpp, salve-o em um diretório juntamente com o arquivo CMakeLists.txt adaptado para o exemplo (Listagem 26, “CMakeLists.txt”), com as imagens biel.png e marcadagua.png e execute a seqüência de comandos mostrada na Listagem 27, “Compilação e execução do programa combinacao”.

Listagem 26. CMakeLists.txt
cmake_minimum_required(VERSION 3.0.0)
project(combinacao VERSION 0.1.0 LANGUAGES C CXX)

find_package(OpenCV REQUIRED)
include_directories(${OpenCV_INCLUDE_DIRS})

add_executable(combinacao combinacao.cpp)
target_link_libraries(combinacao ${OpenCV_LIBS})
Listagem 27. Compilação e execução do programa combinacao
$ mkdir build
$ cd build
$ cmake ..
$ make
$ ./combinacao ../biel.png ../marcadagua.png

8.1. Descrição do programa combinacao.cpp

img_add = image1 + image2;
img_sub = cv::abs(image1 - image2);
img_max = cv::max(image1, image2);
img_min = cv::min(image1, image2);
cv::addWeighted(image1, 1.0, image2, 0.3, 0, img_addweighted);

As quatro linhas realizam as operações de soma, subtração, máximo e mínimo entre as imagens image1 e image2. A função cv::addWeighted realiza a soma ponderada das imagens image1 e image2 com pesos 1.0 e 0.3, respectivamente, usando a equação (img_addweighted = image1 * 1.0 + image2 * 0.3). O resultado de cada operação é armazenado em uma imagem de saída. Perceba que a operação de subtração é feita jutamente com a função cv::abs, que calcula o valor absoluto da diferença entre os pixels das imagens, para evitar valores negativos.

cv::hconcat(img_add, img_sub, comb);

A função cv::hconcat concatena as imagens img_add e img_sub horizontalmente e armazena o resultado na imagem comb. Essa função é utilizada para juntar as imagens de saída em uma única imagem, algo bastante comum em aplicações de visão computacional quando se deseja visualizar o resultado de várias operações.

cv::bitwise_and(image1, image2, img_and);
cv::bitwise_or(image1, image2, img_or);
cv::bitwise_not(image1, img_not);
cv::bitwise_xor(image1, image2, img_xor);

As quatro linhas realizam as operações lógicas AND, OR, NOT e XOR entre as imagens image1 e image2. O resultado de cada operação é armazenado em uma imagem de saída. As operações bit-a-bit são realizadas nos bytes de cada pixel das imagens, o que permite a criação de efeitos interessantes.

Para realizar as operações de combinação de imagens, é necessário que as imagens de entrada tenham o mesmo tamanho e número de canais. Caso contrário, o OpenCV irá lançar uma exceção.
