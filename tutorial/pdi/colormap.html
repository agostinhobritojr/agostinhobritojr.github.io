Capítulo 6. Colorizando imagens cinzentas com mapas de cores

O objetivo dessa lição é mostrar como colorizar imagens cinzentas usando mapas de cor, ou colormaps.

Colormaps (ou Look-Up Tables - LUT) são tabelas de cores que mapeiam valores de intensidade de colormap para cores. Elas são comumente usadas para representar dados de intensidade em imagens em tons de cinza, como mapas de elevação, imagens médicas, entre outros, onde a cor destaca a intensidade do valor com maior riqueza visual.

O exemplo da Listagem 19, “colormap.cpp” abre uma imagem e a exibe utilizando um dos mapas de cores pré programados do OpenCV.

Listagem 19. colormap.cpp
#include <opencv2/opencv.hpp>

int main(int argc, const char *argv[]){
  cv::Mat image, imagecolor;
  
  image = imread(argv[1], cv::IMREAD_GRAYSCALE);
  if(!image.data){
    std::cout << "nao abriu a imagem" << std::endl;
  }

  cv::applyColorMap(image, imagecolor, cv::COLORMAP_JET);
  cv::imshow("colormap_jet", imagecolor);
  cv::waitKey(0);
  return 0;
}

Para compilar e executar o programa-exemplo colormap.cpp, salve-o em um diretório juntamente com o arquivo CMakeLists.txt adaptado para o exemplo (Listagem 20, “CMakeLists.txt”), a imagem biel.png e execute a seqüência de comandos mostrada na Listagem 21, “Compilação e execução do programa colormap”.

Listagem 20. CMakeLists.txt
cmake_minimum_required(VERSION 3.0.0)
project(colormap VERSION 0.1.0 LANGUAGES C CXX)

find_package(OpenCV REQUIRED)
include_directories(${OpenCV_INCLUDE_DIRS})

add_executable(colormap colormap.cpp)
target_link_libraries(colormap ${OpenCV_LIBS})
Listagem 21. Compilação e execução do programa colormap
$ mkdir build
$ cd build
$ cmake ..
$ make
$ ./colormap ../biel.png

6.1. Descrição do programa colormap.cpp

cv::applyColorMap(image, imagecolor, cv::COLORMAP_JET);

A função cv::applyColorMap aplica um mapa de cores a uma imagem de entrada. O primeiro argumento é a imagem de entrada, o segundo argumento é a imagem de saída e o terceiro argumento é o mapa de cores a ser aplicado. O OpenCV fornece vários mapas de cores, como cv::COLORMAP_JET, cv::COLORMAP_PARULA, cv::COLORMAP_SPRING, entre outros.

Os mapas devem ser escolhidos de acordo com a aplicação e o tipo de imagem que se deseja apresentar e, apesar de serem úteis para destacar detalhes em imagens em tons de cinza, é importante lembrar que a cor não deve ser usada como única informação para a interpretação de dados e que o dado mascarado pela cor ainda é a imagem em tons de cinza.
