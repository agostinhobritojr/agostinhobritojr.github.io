Capítulo 22. Extração de características: Momentos de Hu para regiões

Os momentos de Hu são um conjunto de sete características globais que podem ser calculadas a partir de uma imagem ou de um momentos. Eles são invariantes a operações de translação, rotação, escala e espelhamento, o que os torna úteis para a extração de características de forma. Eles são amplamente utilizados em aplicações de reconhecimento de padrões, com bons resultados para formas geométricas simples, dada sua rapidez e simplicidade de cálculo.

Em se tratando de regiões, eles são calculados a partir dos momentos de uma imagem, consistem em combinações de somas intensidades dos pixels ponderadas pelas posições que ocupam no espaço. Em se tratando de uma imagem, seus momentos são calculados usando equação

\[m_{pq} = \sum_{x} \sum_{y} x^p y^q I(x, y)\]

onde \(I(x, y)\) é a intensidade do pixel na posição \((x, y)\) da imagem, e \(p\) e \(q\) são inteiros não negativos. O momento central de ordem \(p+q\) é definido como

\[\mu_{pq} = \sum_{x} \sum_{y} (x - \bar{x})^p (y - \bar{y})^q I(x, y)\]

Os momentos centrais normalizados são definidos como

\[\eta_{pq} = \frac{\mu_{pq}}{\mu_{00}^{1 + \frac{p + q}{2}}}\]

A partir dessas equações, Hu propôs um conjunto de sete momentos invariantes, que são calculados a partir dos momentos normalizados

\[\begin{align*} \eta_{00} & = m_{00} \\ \eta_{01} & = m_{01}/m_{00} \\ \eta_{10} & = m_{10}/m_{00} \\ \eta_{11} & = m_{11}/m_{00} \\ \eta_{20} & = m_{20}/m_{00} \\ \eta_{02} & = m_{02}/m_{00} \\ \eta_{30} & = (m_{30} - 3m_{12})/{m_{00}^{3/2}} \\ \eta_{03} & = (m_{03} - 3m_{21})/{m_{00}^{3/2}}\\ \end{align*}\]

Os momentos invariantes de Hu são calculados a partir dos momentos normalizados, conforme as equações

\[\begin{align*} \phi_1 & = \eta_{20} + \eta_{02} \\ \phi_2 & = (\eta_{20} - \eta_{02})^2 + 4\eta_{11}^2 \\ \phi_3 & = (\eta_{30} - 3\eta_{12})^2 + (3\eta_{21} - \eta_{03})^2 \\ \phi_4 & = (\eta_{30} + \eta_{12})^2 + (\eta_{21} + \eta_{03})^2 \\ \phi_5 & = (\eta_{30} - 3\eta_{12})(\eta_{30} + \eta_{12})[(\eta_{30} + \eta_{12})^2 - 3(\eta_{21} + \eta_{03})^2] + (3\eta_{21} - \eta_{03})(\eta_{21} + \eta_{03})[3(\eta_{30} + \eta_{12})^2 - (\eta_{21} + \eta_{03})^2] \\ \phi_6 & = (\eta_{20} - \eta_{02})[(\eta_{30} + \eta_{12})^2 - (\eta_{21} + \eta_{03})^2] + 4\eta_{11}(\eta_{30} + \eta_{12})(\eta_{21} + \eta_{03}) \\ \phi_7 & = (3\eta_{21} - \eta_{03})(\eta_{30} + \eta_{12})[(\eta_{30} + \eta_{12})^2 - 3(\eta_{21} + \eta_{03})^2] - (\eta_{30} - 3\eta_{12})(\eta_{21} + \eta_{03})[3(\eta_{30} + \eta_{12})^2 - (\eta_{21} + \eta_{03})^2] \end{align*}\]

Nesta lição, será mostrado como usar os momentos de Hu para extrair características de contornos e regiões de imagens. Iniciando com o cálculo dos momentos para regiões, os momentos serão usados para identificar características de uma imagem da Figura 56, “Exemplo de imagem para calulo de momentos de Hu.”, que será submetida a diversas transformações geométricas, como rotação, escala, espelhamento e translação.

Figura 56. Exemplo de imagem para calulo de momentos de Hu.

O programa utilizado como referência nesta lição será o da Listagem 69, “momentos-regioes.cpp”. O programa momentos-regioes.cpp abre uma imagem em escala de tons de cinza e extrai os sete momentos de Hu da imagem, mostrando seus valores na saída.

Listagem 69. momentos-regioes.cpp
#include <iostream>
#include <vector>
#include <cmath>
#include <opencv2/opencv.hpp>

int main(int argc, char* argv[]) {
  double huMoments[7];
  
  cv::Mat image = cv::imread(argv[1], cv::IMREAD_GRAYSCALE);

  cv::Moments moment = cv::moments(image, false);
  
  cv::HuMoments(moment, huMoments);

  for (int i = 0; i < 7; i++) {
    std::cout << huMoments[i] << ", ";
  }
  std::cout << std::endl;

  for (int i = 0; i < 7; i++) {
    huMoments[i] = -1 * std::copysign(1.0, huMoments[i]) * log10(abs(huMoments[i]));
  }

  for (int i = 0; i < 7; i++) {
    std::cout << huMoments[i] << ", ";
  }

  std::cout << std::endl;
  return 0;
}

Perceba que, embora as imagens sejam diferentes, os valores dos logaritmos dos momentos de Hu são praticamente os mesmos. Isso ocorre porque os momentos de Hu são praticament invariantes às transformações geométricas às quais foi submetida a primeira imagem. 22.1. Descrição do programa momentos.cpp

cv::Moments moment = cv::moments(image, false);

cv::HuMoments(moment, huMoments);

A função cv::moments() calcula os momentos da imagem fornecida e retorna um objeto do tipo cv::Moments que armazena os momentos da imagem. A função cv::HuMoments() calcula os momentos de Hu a partir dos momentos da imagem e armazena os valores dos momentos de Hu no array huMoments.

for (int i = 0; i < 7; i++) {
  huMoments[i] = -1 * std::copysign(1.0, huMoments[i]) * log10(abs(huMoments[i]));
}

Neste exemplo, o cálculo dos logaritmos dos momentos de Hu é realizado para permitir melhor comparação em escala, posto que podem variar muito dependendo da imagem, dificultando a comparação. Ainda, o cálculo dos logaritmos dos momentos de Hu é uma prática comum em aplicações de reconhecimento de padrões, pois permite uma interpretação mais intuitiva dos valores com relação às propriedades geométricas da região.
