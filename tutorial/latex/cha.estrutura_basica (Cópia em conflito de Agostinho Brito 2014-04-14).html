Nesse capítulo mostraremos como criar uma documento em LaTeX, começando da estrutura bem simples e evoluindo até a criação de um documento mais elaborado. Essa estrutura é apresentada no Exemplo 2.1.

Exemplo 2.1. Estrutura Básica

\documentclass[a4paper,12pt]{article}
\begin{document}
Esse é um exemplo de como se usa alguns comandos básicos para criação de documentos em \LaTeX.
\end{document}

Este exemplo é composto de quatro linhas e quando compilado uma única página é produzida no documento de saída, como mostra a Figura 2.1, mas antes de compilar vamos entender o que cada comando significa.

2.1. Comandos Básicos

A primeira linha do documento contém as declarações

\documentclass[a4paper,12pt]{article}

Nesse comando definimos que tipo de documento queremos criar e as configurações fundamentais do documento, tais como o tamanho do papel e o tamanho da fonte. Neste caso, possíveis classes e opções a serem passadas para esse comando documentclass são:

Classes

article

Essa classe configurar o documento para forma de um artigo.

report

Essa classe é usada para relatórios longos.

book

Essa classe deve ser utilizada quando se quer criar livro.

slides:

Essa classe configura seu documento como um Slide.

letter

Essa classe é ideal para criação de carta.

Opções

  1. Tipo de Papel: essa opção nos dar vários tipos de papeis para configurar nosso documento, são elas letterpaper (formato de carta) (215,9mm X 279,4mm), legalpaper (formato legal) esse é um tipo de papel econtrado no EUA (216mm X 356mm), executivepaper (formato executivo) usado para cartas oficiais (184mm X 267mm), a4paper (formato A4) dimensões (210mm X 297mm) e a5paper (formato A5) dimensões (148mm X 210mm), entre outros tipos, porém nesse tutorial utilizaremos o formato a4paper.

  2. Tamanho da Fonte: nessa opção podemos alterar o tamanho da fonte de todo o documento, sendo disponível somente três tamanhos 10pt, 11pt e 12pt. obs: para alterar o tamanho de uma determinada palavra, frase ou até mesmo um texto usa-se outra tag que mostraremos mais a adiante.

  3. Orientação do Documento: com essa opção podemos alterar a orientação da página do documento da vertical para horizontal usando a opção landscape.

  4. Impressão: com essa opção definimos como vai ser a impressão do documento se é em um lado (oneside) da folha ou nos dois lados (twoside), caso não seja declarada a impressão será definida como oneside com exceção da classe book que usa como padrão o twoside.

  5. Colunas: essa opção nos dar a alternativa de dividir a página em duas colunas usando o twocolumn exceto na classe slides.

  6. Equações: exite uma opção que faz com que as equações apresentadas no documento fiquem alinhadas à esquerda (fleqn) em vez do centro e uma opção que coloca a numeração das equações tambem à esquerda (leqno) ao invés da direita.

Existem outras opções além dessas, porém essas são as principais opções que iremos utilizar durante o tutorial.

Na segunda e quarta linha do Exemplo 2.1 temos os respectivos comandos 

\begin{document}

\end{document}

É entre esses dois comandos que criaremos o nosso texto.

Para entender melhor os recursos que serão necessários para que o texto fique correto, é importante compreender que recursos são esses e o que fazem no seu documento.

Os recursos que utilizaremos serão os Pacotes 

\usepackage[opção]{pacote}

Eles devem ser declarados logo após o comando 

\documentclass[]{}

e antes do comando 

\begin{document}

Com esse comando você pode importar pacotes específicos para configuração do seu documento, mostraremos alguns pacotes mais utilizados.

Pacotes

  1. \usepackage[brazil]{babel}

    Esse pacote configura seu texto para a forma brasileira, reconhecendo hífenização e outras coisas.

  2. \usepackage[utf8]{inputenc}

    Com esse pacote o latex reconhece acentos e caracteres especiais no seu texto.

  3. \usepackage{color}

    Esse pacote disponibiliza a opção de mudar as cores das fontes do seu texto.

  4. \usepackage{graphicx}

    Esse pacote é utilizado quando você precisar adicionar imagens ao seu documento.

  5. \usepackage{amsfonts}

    Ao incluir esse pacote você pode alterar o estilo das fontes utilizadas no ambiente matemático.

  6. \usepackage{fancyhdr}

    Com esse pacote você pode personalizar o cabeçalho e o rodapé do seu documento.

  7. \usepackage[normalem]{ulem}

    Esse pacote é usado quando precisa de sublinha alguma palavra.

  8. \usepackage{indentfirst}

    Com esse pacote a primeira linha de todas as seções e subseções são recuadas para o parágrafo usual.

2.2. Comandos para editoração

Agora veremos por ordem hierárquica alguns comandos necessários para organização do documento em capítulos, seções, subseções e etc.

O primeiro comando da hierárquia é o 

\chapter{título}

ao usa esse comando você cria capítulos numerados.

Atenção

o comando \chapter{título} é exclusivo para as classes book e report, já os comandos listado abaixo serve tanto para book e report como para as demais classes.

Em seguida temos o comando 

\part{título}

Com ele criaremos textos particionados como por exemplo: Parte I, Parte II e etc.

Logo após temos 

\section{título}

Esse comando cria seções numeradas.

Para criarmos uma subseção numerada utilizamos 

\subsection{título}

Se quisermos criar uma subseção numerada dentro de outra subseção, utilizamos o 

\subsubsection{título}

Temos também o comando 

\paragraph{título}

usado para criar parágrafos.

E por último na hieráquia temos o 

\subparagraph{título}:

que cria subparágrafos.

Importante

Você pode retirar as numerações e deixar só os titulos colocando um "*" entre o comando e o abre chaves "{" como mostrado abaixo.

\section*{titulo}

Depois de utilizado os comandos de seção, subseção e os demais, podemos agora criar uma súmairo usando o 

\tableofcontents

onde ele cria automaticamente o súmario de acordo com a utilização de \section{título}, \subsection{título}, \subsubsection{título}, \part{título} e \chapter{título} que você estiver fazendo.

Dica

É recomendável você colocar o comando \newpage logo após o comando \tableofcontents, para que o súmario fique separado do inicio do texto.

Agora que conhecemos mais alguns comandos do LaTeX vamos colocar em prática o que aprendemos como mostra o Exemplo 2.3.

Exemplo 2.3. Documento com seções e etc.

\documentclass[a4paper,12pt]{article}
\usepackage[brazil]{babel}
\usepackage[utf8]{inputenc}
\begin{document}
\tableofcontents
\newpage
\part{título}
\section{Comandos para editoração do texto}
Nesse exemplo mostraremos como usar os comandos para editoração do texto, como por exemplo essa seção.
\subsection{Subseção}
Agora criamos uma subseção.
\subsubsection{Subseção dentro de uma subseção} 
Agora temos uma subseção dentro de outra.
\paragraph{Parágrafo}
Aqui temos um parágrafos.
\subparagraph{Subparágrafo}
E para finalizar temos um subparágrafo.
\end{document}

Ao compilar Exemplo 2.3 o documento da Figura 2.3 será gerado.

Figura 2.3. Documento com seções e etc


Nota

Os comandos de parágrafos e subparágrafos não são muito utilizados, uma vez que pode-se criar parágrafos sem a necessidade de comandos.

Nota

Talvez na primeira compilação do seu documento o súmario não apareça por completo, para resolver este problema é so compilar novamente e ele aparecera.
