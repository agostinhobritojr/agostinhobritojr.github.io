Agora apresentaremos os ambientes existentes no $\textrm{\LaTeX}$ e como são utilizados.

Um ambiente é criado ao usarmos os comandos 

\begin{ambiente} texto aqui \end{ambiente}

onde no campo ambiente nos iremos atribuir um dos ambientes que serão mostrados ao decorrer deste capítulo.

3.1. Listas

Existem três ambientes que poderemos utilizar para criar listas, são elas:

itemize

Esse ambiente é usado quando queremos criar uma lista simples com qualquer tipo de símbolo.

Para criarmos itens nesse ambiente utilizamos o comando 

\item[símbolo]

onde o símbolo pode ser desde letras até alguns caracteres especiais.

enumerate

Esse outro ambiente é utilizado quando queremos criar uma lista numerada, assim como o itemize utilizamos o comando 

\item

para criar os itens da lista.

description

Esse ambiente utilizamos para criar uma lista com descrição, utilizamos o comando 

\item[título]

para criar um item com o título a ser descrito.

Vejamos uma aplicação desse ambiente no Exemplo 3.1.

Exemplo 3.1. Criando Listas

\documentclass[a4paper,12pt]{article}
\usepackage[brazil]{babel}
\usepackage[utf8]{inputenc}
\begin{document}
\begin{itemize}
\item Esse é um item sem numeração.
\begin{enumerate}
\item Já esse possui numeração.
\item perceba que é possível criar um ambiente dentro de outro ambiente.
\end{enumerate}
\begin{description}
\item[Título]Aqui colocamos a descrição desse item.
\end{description}
\end{itemize}
\end{document}

3.2. Tabelas

Para criarmos tabelas no $\textrm{\LaTeX}$ utilizaremos o ambiente 

\begin{tabular}[posição]{colunas}

onde os parâmetros da tabela serão passados nos campos posição e colunas.

Posição

Nesse campo colocaremos a posição vertical do conteúdo da tabela com relação a linha da tabela, sendo elas t(topo), b(baixo) e c(centro).

Colunas

Esse campo diferente do campo posição, serve não só para indicar o alinhamento horizontal do conteúdo da tabela como também serve para indicar a quantidade de colunas que queremos na tabela, sendo os alinhamentos, l(esquerda), r(direita) e c(centro).

Para definirmos a quantidade de colunas da tabela, utilizaremos a quantidade de letras que defini o alinhamento, ou seja, se quisermos uma tabela com três colunas e alinhamento no centro colocaremos 

\begin{tabular}{c c c}

Fazendo isso dizemos ao $\textrm{\LaTeX}$ que queremos uma tabela com três colunas centralizadas e sem bordas verticais, para introduzir as bordas verticais utilizamos as duas barras "| |" e dentro das duas barras colocaremos um dos alinhamentos, podendo colocar cada coluna com alinhamento diferente se preferir.

Podemos também definir a largura de uma coluna em centímetros, basta colocar p{largura cm} em vez do alinhamento.

Dentro do ambiente da tabela iniciaremos colocando o comando 

\hline

que desenha a primeira borda horizontal da tabela em seguida colocaremos o conteúdo de cada coluna separado por um "&", logo após colocaremos "\\" para pular para proxima linha da tabela e pra finaliza colocamos novamente o comando \hline para desenhar a proxima borda vertical. Vejamos o Exemplo 3.2.

Exemplo 3.2. Criando uma Tabela Simples

\documentclass[a4paper,12pt]{article}
\usepackage[brazil]{babel}
\usepackage[utf8]{inputenc}
\begin{document}
\begin{tabular}{|c|p{2.5 cm}|}
\hline
\textbf{Nome} & \textbf{Sobrenome}\\\hline
Fulano & do Cicranal \\\hline
Beltrano & Fulanal \\\hline 
\end{tabular}
\end{document}

No $\textrm{\LaTeX}$ também podemos mesclar células com o comando 

\multicolum{nº de colunas}{alinhamento}{conteúdo}

onde no campo nº de colunas colocaremos o numero de colunas que queremos mesclar.

No campo alinhamento colocaremos um dos três comandos já visto (l, c ou r).

Outra ferramenta que o $\textrm{\LaTeX}$ nos dá para formatação de tabelas é a de linhas parciais, ao utilizamos o comando 

\cline{a-b}

onde a e b são os números das colunas de onde e para onde a linha deve ser desenhada. Vejamos o Exemplo 3.3

Exemplo 3.3. Criando uma Tabela

\documentclass[a4paper,12pt]{article}
\usepackage[brazil]{babel}
\usepackage[utf8]{inputenc}
\begin{document}
\begin{tabular}{|c|c|}
\hline
\multicolumn{2}{|c|}{\textbf{Dados Pessoais}}\\\hline
\textbf{Nome} & \textbf{Sobrenome}\\\hline
Fulano & do Cicranal \\\cline{1-1}
Beltrano &  \\\hline 
\end{tabular}
\end{document}
