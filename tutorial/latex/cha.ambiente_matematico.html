No $\textrm{\LaTeX}$ existem vários ambientes que possuem características específicas como por exemplo, tabelas, listas entre outros.

Nesse capítulo veremos o ambiente matemático, onde mostraremos como fazer equações utilizando uma gama de símbolos matemáticos disponiveis no $\textrm{\LaTeX}$.

4.1. Equações

Para criarmos o ambiente matemático na linha onde possui texto podemos utilizar duas formas diferentes, entre $ e $ ou entre 

\begin{math} equação \end{math}

Para colocarmos equações maiores em um parágrafo separado do texto, utilizaremos o 

\begin{displaymath} equação \end{displaymath}

Se quisermos além de colocar a equação em um parágrafo separado, numera-la, o comando a ser utilizado é o 

\begin{equation} equação \end{equation}

Agora vejamos o Exemplo 4.1 com algumas equações utilizadas no meio acadêmico

Exemplo 4.1. Criando Equações

\documentclass[a4paper,12pt]{article}
\usepackage[brazil]{babel}
\usepackage[utf8]{inputenc}
\begin{document}
Teorema de pitágoras:\begin{displaymath}
h^2 = a^2 + b^2 \rightarrow h=\sqrt{a^{2}+b^{2}}
\end{displaymath}
Série de Taylor: \begin{displaymath}
f(x) =\sum_{n=0}^{\infty}\frac{f^{(n)}(a)(x-a)^n}{n!}
\end{displaymath} 
Derivada: \begin{displaymath}f'(x) = \frac{df}{dx}=\lim_{x\rightarrow a}\frac{f(x)-f(a)}{x-a} 
\end{displaymath}
Série de Fourier: \begin{displaymath}
f(t)=\sum_{n=-\infty}^{\infty}c_n\exp^{\frac{i\pi n t}{L}};\quad\textrm{onde}\quad 
c_n= \frac{1}{2L}\int_{c}^{c+2L}f(t)\exp ^{-\frac{i\pi n t}{L}}\mathrm{dt}
\end{displaymath}
\end{document}

Ao compilarmos esse código as equações mostradas na Figura 4.1 serão geradas

Figura 4.1. Criando Equações


Ao visualizar o Exemplo 4.1 percebemos que existem alguns comandos que ainda não tínhamos visto, que vão desde comandos de operações matemáticas até comando de símbolos matemáticos, esses serão apresentados a seguir.

4.1.1. Potência, Raíz e Índice

Potência

Para criarmos uma equação com potência utilizamos o circunflexo (^) entre a base e o expoente (B^e), caso o expoente tenha mais de um termo utilizaremos as chaves { } (B^{e+x}).

Raíz

Para criarmos uma equação com raíz, utilizaremos o comando 

\sqrt[ordem]{expressão}

onde no campo "ordem" colocaremos a ordem da raíz e no campo "expressão" colocaremos a expressão.

Índice

Podemos criar também índices facilmente utilizando o underline (_) entre o coeficiente e o índice que queremos dar a esse coeficiente (c_1, c_2, ...etc.).

Vejamos o Exemplo 4.2

Exemplo 4.2. Usando Potências, Raízes e Índices

\documentclass[a4paper,12pt]{article}
\usepackage[brazil]{babel}
\usepackage[utf8]{inputenc}
\begin{document}
Teorema de pitágoras:\begin{displaymath}
h=\sqrt[]{a_1^{2}+a_2^{2}}
\end{displaymath}
\end{document}

4.1.2. Fração, Integral e Derivada

Fração

No $\textrm{\LaTeX}$ podemos criar facilmente uma fração utilizando um dos dois comandos abaixo 

\frac{numerador}{denominador}

ou 

\sfrac{numerador}{denominador}
Atenção

O comando \sfrac{numerador}{denominador} para funcionar, necessita que importemos o pacote \usepackage{xfrac}

Integral

O comando que utilizaremos para criar uma integral é o 

\int_{limite inferior}^{limite superior}
Dica

Para fazermos integrais múltiplas basta adicionas a letra "i" depois da \ e antes do int e adicionar o pacote \usepackage{amsmath}.

Derivada

No $\textrm{\LaTeX}$ podemos representar a derivada por 

\frac{df}{dx}

ou simplesmente colocando f'(x).

Podemos fazer também derivadas parciais com o seguinte comando 

\frac{\partial f}{\patial x}
Dica

Para alterar a ordem da derivada utilizaremos o "^"

Exemplo 4.3. Usando Frações, Integrais e derivadas

\documentclass[a4paper,12pt]{article}
\usepackage[brazil]{babel}
\usepackage[utf8]{inputenc}
\usepackage{xfrac}
\begin{document}
Frações:\begin{displaymath}
f(x)=\frac{a}{b}=\sfrac{a}{b}
\end{displaymath}
Integrais:\begin{displaymath}
\int_{inf}^{sup} f(x) dx;\quad \int_{0}^{x} \int_{0}^{y} f(x,y)dx dy
\end{displaymath}
Derivadas:\begin{displaymath}
f'(x)=\frac{df}{dx};\quad f''(x)=\frac{d^2f}{dx^2};\quad \frac{\partial ^2 f}{\partial x^2}
\end{displaymath}
\end{document}

4.1.3. Somatório, Produtório e Matriz

Somatório

O somatório pode ser criado seguindo a mesma estrutura da integral modificando apenas o comando 

\sum_{limite inferior}^{limite superior}
Produtório

O produtorio segue o mesmo padrão do somatório 

\prod_{limite inferior}^{limite superior}
Matriz

Para criamos uma matriz usaremos outro ambiente dentro do ambiente matemático 

\begin{array}{colunas} matriz \end{array}

onde no campo "colunas" colocaremos a quantidade de colunas que vai ter nossa matriz sendo elas definidas pela quantidade de letra "c", para dividirmos os elementos de cada coluna utilizamos o "&" e para pular para próxima linha da matriz utilizamos "\\" como mostra o Exemplo 4.4

Exemplo 4.4. Usando Somatório, Produtório e Matriz

\documentclass[a4paper,12pt]{article}
\usepackage[brazil]{babel}
\usepackage[utf8]{inputenc}
\begin{document}
Somatório:\begin{displaymath}
f(x) =\sum_{n=0}^{\infty}\frac{f^{(n)}(a)(x-a)^n}{n!}
\end{displaymath}
Produtório:\begin{displaymath}
\prod_{i=1}^{n}Ri
\end{displaymath}
Matriz:\begin{displaymath}
\left[
\begin{array}{ccc}
1 & 2 & 3 \\
4 & 5 & 6 \\
7 & 8 & 9 \\
\end{array}
\right]_{3\times 3}
\end{displaymath}
\end{document}

4.1.4. Limite

Para utilizarmos o limite no $\textrm{\LaTeX}$ devemos seguir a seguinte estrutura de código 

\lim_{x \rightarrow a} \frac{\sin x}{x}

onde o comando \lim diz para o $\textrm{\LaTeX}$ que utilizaremos o limite, logo após colocamos um "_" para definimos qual será esse limite colocando o comando {x \rightarrow a}, e por último colocamos a equação \frac{\sin x}{x}.

Exemplo 4.5. Usando Limite

\documentclass[a4paper,12]{article}
\usepackage[brazil]{babel}
\usepackage[utf8]{inputenc}
\begin{document}
\begin{displaymath}
\lim_{x \rightarrow 0} \frac{\sin x}{x} = 1
\end{displaymath}
\end{document}

Figura 4.5. Usando Limite


4.1.5. Vetores

Podemos utilizar os vetores facilmente com um simples comando 

\vec{A}

porém esse comando serve somente para uma caractere, para colocarmos um vetor em mais de uma caractere seguida utilizaremos o comado 

\overrightarrow{AB}

Exemplo 4.6. Usando Vetores

\documentclass[a4paper,12]{article}
\usepackage[brazil]{babel}
\usepackage[utf8]{inputenc}
\begin{document}
\begin{displaymath}
\vec{w}=\vec{v}+\vec{u}
\end{displaymath}
\end{document}

4.3. Referências Cruzadas (Label)

Comumente utilizamos referências cruzadas para facilitar a leitura de um artigo ou relatório entre outros, no $\textrm{\LaTeX}$ não seria diferente, através do comando 

\label

podemos utilizar a numeração do ambiente equation para referenciar a equação em qualquer lugar do texto, para isso devemos colocar o comando 

\label{id}

logo após o comando 

\begin{equation}

onde no campo "id", colocaremos uma etiqueta para sabermos qual equação vamos referenciar.

Para chamarmos a referência da equação no texto, devemos seguir o seguinte esquema 

(\ref{id})

Vejamos o Exemplo 4.7

Exemplo 4.7. Usando Label

\documentclass[a4paper,12]{article}
\usepackage[brazil]{babel}
\usepackage[utf8]{inputenc}
\begin{document}
\begin{equation} \label{linear}
f(x)=ax+b
\end{equation}
Esse é um simples exemplo de referências cruzadas\\
Observe a equação (\ref{linear}). 
\end{document}

Figura 4.7. Usando Label


Nota

Algumas vezes podem aparecer interrogações no lugar da referência, caso isso aconteça verifique se a etiqueta esta correta, caso esteja correta é só compilar novamente e a referência aparecerá.
