Capítulo 2. Criando um documento básico em Latex

Índice

2.1. Comandos:

Nesse capítulo mostraremos como criar uma documento em LaTeX, começando da estrutura básica apresentada no Exemplo 2.1 até a criação de um relatório básico e esmiunçando cada comando apresentado.

Exemplo 2.1. Estrutura Básica

\documentclass[12pt,a4paper]{article}
\begin{document}
Olá mundo.
\end{document}

antes de compilar esse documento vamos entender o que foi feito.

2.1. Comandos:

\documentclass[opções]{classes}: Nesse comando definimos que tipo (classes) de documento queremos criar e as configurações (opções) do documento, são elas:

Classes

  1. article: essa classe configurar o documento para forma de um artigo.

  2. report: essa classe é usada para relatórios longos.

  3. book: essa classe deve ser utilizada quando se quer criar livro.

  4. slides: essa classe configura seu documento como um Slide.

  5. letter: essa classe é ideal para criação de carta.

Opções

  1. Tipo de Papel: essa opção nos dar vários tipos de papeis para configurar nosso documento, são elas letterpaper (formato de carta) (215,9mm X 279,4mm), legalpaper (formato legal) esse é um tipo de papel econtrado no EUA (216mm X 356mm), executivepaper (formato executivo) usado para cartas oficiais (184mm X 267mm), a4paper (formato A4) dimensões (210mm X 297mm) e a5paper (formato A5) dimensões (148mm X 210mm), entre outros tipos, porém nesse tutorial utilizaremos o formato a4paper.
