Capítulo 6. Elaborando Referências

Para criarmos uma bibliografia, utilizamos o ambiente 

\begin{thebibliography}{numero_maximo}

onde no "numero_maximo" definimos a quantidade máxima de referências que podemos utilizar.

Dentro desse ambiente podemos definir varias referencias utilizando para cada uma delas o comando 

\bibitem{id}

onde no campo id colocamos uma etiqueta para podermos citar essa referência durante o texto utilizando o comando 

\cite{id}

Exemplo 6.1. Criando Referências

\documentclass[a4paper,12pt]{article}
\usepackage[brazil]{babel}
\usepackage[utf8]{inputenc}
\begin{document}
\begin{thebibliography}{5}
\bibitem{pa} H.~Partl: \emph{German \TeX}, TUGboat Volume~9, Issue~1 (1988)
\end{thebibliography}
\end{document}
